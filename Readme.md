## 📝 Assignment Questions & Answers

### 1. Difference between `null` and `undefined`
- **Undefined**: Variable declared but no value assigned.  
- **Null**: Variable intentionally assigned an empty value.

### 2. `map()` vs `forEach()`
- **`map()`**
  - Creates a **new array**  
  - Transforms each element  
- **`forEach()`**
  - Loops through array elements  
  - **Does not return a new array**  
- **Difference**
  - `map()` → returns **new array**  
  - `forEach()` → returns **undefined**

### 3. `==` vs `===`
- `==` → Compares **value only** (type can differ)  
- `===` → Compares **value and type** (must match)

### 4. Significance of `async/await`
- Simplifies asynchronous code  
- Avoids long `.then()` chains  
- Easier error handling with `try...catch`  
- Waits for API response before moving to the next line

### 5. JavaScript Scope
- **Global Scope** → Accessible anywhere  
- **Function Scope** → Accessible only inside function  
- **Block Scope** → Accessible only inside `{}` (with `let` or `const`)
